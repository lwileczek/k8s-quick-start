# Vagrant Cluster

Deploy a kubernetes deployment on three virtual machines

## Deployment

  - Kubelet
  - Kubeadm
  - Kubectl
  - Calico

## Assumptions
 - Vagrant is installed
 - Internet Connection 
 - 8GB of Memory
 - 4 cores
